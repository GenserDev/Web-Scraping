<h1 align="center">
Web Scraping
</h1>

<p align="center">
  En esta actividad, aplicarán conceptos de buffers, centinelas y expresiones regulares para realizar scraping en páginas web. El objetivo es encontrar las imágenes de productos en una tienda de comercio electrónico o de videojuegos y generar un listado exportable de los productos con sus nombres e imágenes.
</p>

## Como compilar
```
py Scrapping
```

> Para fines educativos la tienda utilizada para este ejercicio fue la tienda Max de Guatemala, especificamente su sección de juegos de la consola Switch 2.
